## 2. Purpose

The Jungus Professional Test and Jungus Standard Test are Chinese cognitive-function assessments developed by the Jungus team.

The professional version is designed to measure fine-grained differences within traditional 16-type personality categories and supports the Jungus 16-function and 64-archetype model.

The standard version is a lower-load assessment intended for easier public use while preserving the core cognitive-function measurement direction.

## 3. Theoretical Structure

The Jungus model keeps the classical MBTI and Jungian function vocabulary as a reference point, but extends it by introducing two additional preference dimensions:

- Object-Background preference for perceiving functions.
- Analysis-Holism preference for judging functions.

These dimensions produce 16 Jungus cognitive functions and 64 personality archetypes.
